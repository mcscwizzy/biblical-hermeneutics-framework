# Examples

Walkthroughs that show the BHF method in action. Each example demonstrates
**how** to interpret responsibly — it does **not** settle the doctrinal question.

- [`romans-honor-shame-walkthrough.md`](romans-honor-shame-walkthrough.md) —
  applying the core method + epistle + Roman-setting modules to a passage in
  Romans, modeling genre-awareness, confidence labels, and multiple views.

To reproduce an example, compose the modules it names and run it through any
model (see [`../docs/how-to-use/`](../docs/how-to-use/)).
