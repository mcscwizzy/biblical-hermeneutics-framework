# Golden responses

Reference annotations of good and poor model responses for selected fixtures in
[`../prompts/`](../prompts/), used to calibrate human scoring against the
[rubrics](../rubrics/). These annotate *method* (did the response identify genre
first, label confidence, present multiple views, avoid fabrication?), never the
correctness of a doctrinal conclusion.

This folder is seeded as the behavioral suite grows. To contribute a golden
annotation, capture a model's response to a fixture and mark which
[`core-behaviors`](../rubrics/core-behaviors.yml) criteria it met and why.
